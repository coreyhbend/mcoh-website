# Plugin Setup Checklist

## Core Plugins

### SEO
- RankMath Pro or Yoast SEO

### Performance
- WP Rocket
- Cloudflare

### Content Architecture
- Advanced Custom Fields Pro
- The Events Calendar

### Feed Aggregation
- WP RSS Aggregator

### Forms and Newsletter
- Gravity Forms

### Media Optimization
- Cloudinary or Cloudflare Images

### Backup
- UpdraftPlus

## Initial Configuration Priorities

1. Configure permalinks
2. Configure sitemap
3. Configure schema
4. Configure image compression
5. Configure caching
6. Configure feed imports
7. Configure event imports
8. Configure backups
9. Configure security settings

## Recommended Theme Settings

- Disable unnecessary plugins
- Avoid heavy page builders
- Prioritize performance and editorial layout quality
- Use WebP images throughout
- Keep typography and spacing consistent with brand guidelines
