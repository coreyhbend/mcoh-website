# Deployment Checklist

## Goal

Prepare Make Central Oregon Home for a staging WordPress installation.

## Step 1: Create Staging Site

Recommended options:

- Cloudways
- Rocket.net
- Kinsta
- SiteGround Cloud
- LocalWP for local development

## Step 2: Install WordPress

After WordPress is installed:

- Set site title to Make Central Oregon Home
- Set tagline to Central Oregon Lifestyle Guide
- Set permalink structure to Post name
- Disable sample posts and default pages

## Step 3: Upload Theme

Upload this folder as the active theme:

```text
wordpress/themes/mcoh-theme
```

## Step 4: Install Required Plugins

Required:

- Advanced Custom Fields Pro
- RankMath Pro or Yoast SEO
- WP RSS Aggregator
- The Events Calendar
- WP Rocket
- Gravity Forms

Optional:

- Cloudinary
- UpdraftPlus
- Microsoft Clarity

## Step 5: Configure Content Types

After activating the theme, confirm these content types appear in WordPress admin:

- Communities
- Destinations
- Local Intel
- Events
- Breweries

## Step 6: Create Starter Pages

Create these WordPress pages:

- Home
- Explore
- Destinations
- Communities
- Local Intel
- Journal
- Moving to Central Oregon
- About
- Contact

Set Home as the static homepage.

## Step 7: Seed Initial Content

Start with:

- 4 communities
- 4 destinations
- 4 local intel posts
- 4 explore guides

## Step 8: SEO Setup

Configure:

- XML sitemap
- titles and meta descriptions
- organization schema
- website schema
- breadcrumbs
- Open Graph images

## Step 9: Launch Readiness

Before launch:

- Check mobile layout
- Check page speed
- Check broken links
- Check indexed pages
- Submit sitemap
- Connect Google Search Console
- Connect analytics
